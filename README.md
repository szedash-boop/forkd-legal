# forkd-legal

Hosted legal documents for the **FORKD** iOS app, served via GitHub Pages.

- **Terms of Service** — [terms.md](./terms.md)
- **Privacy Policy** — [privacy.md](./privacy.md)
- **Community Guidelines** — [guidelines.md](./guidelines.md)

Live site: <https://szedash-boop.github.io/forkd-legal/>

## Updating

1. Edit the relevant `.md` file.
2. Bump the **Version** string at the top of the page (and at `LegalLinks.currentVersion` in the iOS app if material changes affect consent).
3. Commit + push to `main`. GitHub Pages rebuilds automatically (~1 min).
